# AI Career Prep Assistant

An AI-powered placement preparation assistant built using React, FastAPI, and Gemini AI.

## Features

- Resume ATS analysis
- Skill gap identification
- Personalized learning roadmaps
- AI-powered career guidance
- Interactive frontend using React

## Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- FastAPI
- Python
- Gemini AI API
